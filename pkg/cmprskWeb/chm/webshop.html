webshop(cmprskWeb)R Documentation

Webshop Data

Description

Artificial webshop dataset with page dwell times.

Usage

data(webshop)

Format

A data frame with 20000 observations on the following 6 variables.

session
Session ID
user
User ID
from
Starting page
to
Landing page
time
Dwell time on starting page
buy
buyer vs. non-buyer

Examples

data(webshop)
